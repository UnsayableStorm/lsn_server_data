# LSN Server Data

Configuration data, mod lists, and reference documents for the Lost Space Network (LSN) Space Engineers server.

## Contents

| Path | Description |
|---|---|
| `modlist.csv` | Full server mod list with Workshop IDs |
| `server/torch_import.md` | Workshop IDs grouped by category for Torch bulk import |
| `server/server_notes.md` | General server configuration notes |
| `ships/ship_checklist.md` | Pre-flight ship building checklist |
| `ships/fleet_roster.md` | Known ships and their roles |
| `lcd_configs/odyssey_status.txt` | ShowStats config for Odyssey status panels |
| `lcd_configs/farm_lcd.txt` | InvListNBX farm seed tracking LCD script |
| `lcd_configs/goat_sorter.txt` | HideEmpty category sorter LCD config |
| `mods/` | Notes on custom/special LSN mods |

## Usage

`modlist.csv` can be used with Torch's bulk mod import. The `server/torch_import.md` contains IDs pre-sorted by category for manual Torch paste.
