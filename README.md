# LSN Server Data

Reference data, configs, and mod info for the **Lost Space Network** Space Engineers server.
If you're here, you probably broke something and need a Workshop ID. Welcome.

---

## What's In Here

| File | What it is |
|---|---|
| `modlist.csv` | All 215 mods. Sortable by category. Clickable Workshop links. |
| `engine_data.csv` | Thruster stats — thrust, weight, fuel consumption, T/W ratio. |
| `server/torch_import.md` | Workshop IDs pre-sorted by category for Torch bulk paste. |
| `server/server_notes.md` | Server config notes. |
| `ships/ship_checklist.md` | Pre-flight checklist. Don't skip the gyros. Again. |
| `ships/fleet_roster.md` | Known ships and roles. |
| `lcd_configs/odyssey_status.txt` | ShowStats config for Odyssey status panels. |
| `lcd_configs/farm_lcd.txt` | Farm LCD seed tracker script. |
| `lcd_configs/goat_sorter.txt` | GOAT sorter inventory display config. |
| `mods/` | Notes on LSN custom mods. |

## Modlist

215 mods across 16 categories. Open `modlist.csv` on GitHub to get a filterable table —
click any column header to sort. The **category** column groups by type, **steam_url** links
directly to the Workshop page.

Categories range from the sensible (⚡ Ion thrusters, 🔴 Weapons) to the important
(🌱 Food and Survival, because someone has to feed the crew) to the critical
(🎨 Decorations — the snail plushie is non-negotiable).

---

## Engine Data

Thruster comparison sheet covering all modded and vanilla engines on the server.
Sortable by thrust, weight, fuel type, or T/W ratio. Useful when you're trying to figure out
why your ship flies like a wet sock.

---

## LCD Configs

Copy-paste configs for the LCD scripts running on LSN ships and stations.
Paste into the LCD's script field. If it doesn't work, check that you have the right mod installed.
If it still doesn't work, blame Clang.
